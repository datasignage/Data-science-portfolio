## Прогнозирование коэффициента восстановления золота из руды
### Описание цели проекта
Имеется прмышленное предприятие, добывающее золото. Нам предоставлены данные о физико-химических свойствах всех компонентов всех стадий технологического процесса. Требуется так смоделировать процесс восстановления золота из руды, чтобы, меняя отдельные параметры процесса, отслеживать как меняется целевой показатель - степень очистки золота. В данном проекте требуется создать прототип модели машинного обучения, который показывал бы на реальных данных заданную заказчиком точность предсказания степени очистки.
### Задачи проекта
1. Данные предоставлены *as is*, требуется EDA-анализ с целью формирования единого датасета для обучения и проверки качества моделей.
2. Необходимо написать кастомную метрику (формула расчета степени очистки золота) для оценки качества моделей и их сравнения между собой.
3. Необходимо подобрать модель регрессии обеспечивающую приемлемое качество.

### Выводы
На подготовленных данных были обучены три модели: DecisionTreeRegressor, RandomForestRegressor и LinearRegression, с помощью которой и был получен наилучший приемлемый результат.
